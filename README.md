Задача моего проекта — разработать метод прогнозирования золото-урановых рудных узлов с использованием машинного обучения и нейросетевых подходов. Гипотеза проекта состоит в том, что такие методы могут дать более качественный прогноз по сравнению с классическим критериальным анализом, так как способны учитывать сложные нелинейные сочетания геологических факторов.
